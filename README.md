# Ingrain System

A premium, precision-engineered enterprise ecosystem designed to integrate and elevate every aspect of business operations.

## Features

- **Recruitment Ecosystem**: Automate hiring with precision pipelines.
- **Payroll & HRMS**: Unified command center for workforce management.
- **Lab Management**: Smart tracking for laboratory operations.
- **Coworking & Spaces**: Optimize physical footprints with intuitive logistics.
- **Medical Camps**: Centralized coordination for healthcare outreach.

## Technology Stack

- React
- Tailwind CSS
- Framer Motion
- Lucide React

## Development

```bash
npm install
npm run dev
```
